Mini RPG de batalha por terminal em Node.js, com escolha de personagens, inimigos controlados pelo computador e sistema de combate por turnos.

# MiniRPG Terminal

MiniRPG é um jogo de RPG por terminal desenvolvido em **Node.js**. O jogador escolhe um personagem entre várias classes, enfrenta um adversário controlado pelo computador e batalha até que um dos personagens seja derrotado. O projeto é ideal para estudo de **JavaScript**, **POO** (Programação Orientada a Objetos) e **lógica de jogos**.



## Características

- Sistema de escolha de personagem.
- Classes com atributos diferentes: **Guerreiro, Mago, Arqueiro, Paladino e Assassino**.
- Sistema de combate por turnos.
- Exibição do status de vida de cada personagem.
- Adversário controlado pelo computador.
- Mensagens animadas com pausas para melhor experiência no terminal.



## Tecnologias

- Node.js
- JavaScript ES6
- readline (entrada via terminal)
